# Variation Strategies

When generating 4 variations, maximize visual contrast by picking different combinations from these axes.

**Rule: No two variations may share the same layout strategy.**

## Layout Strategies

Pick 4 different ones per session:

1. **CSS Grid with varied cell sizes** — asymmetric grid, some items span 2 columns or rows
2. **Stacked vertical** — full-width sections stacked, alternating alignment (left/right/center)
3. **Horizontal scroll / carousel** — items in a scrollable row, peek of next item visible
4. **Masonry / bento grid** — Pinterest-style or bento box with mixed aspect ratios
5. **Asymmetric / editorial** — magazine-style layout with overlapping elements, pull quotes, mixed sizing
6. **Split layout** — two-column with content and visual on opposite sides
7. **Centered single-column** — focused, minimal, one item at a time with generous whitespace
8. **Floating cards** — cards with subtle rotation, shadow depth, overlapping edges

## Compositional Approaches

1. **Card-based** — distinct card elements with borders/shadows/backgrounds
2. **Full-bleed sections** — edge-to-edge backgrounds, content overlays
3. **Text-heavy editorial** — large quotes, minimal imagery, typographic hierarchy
4. **Image-dominant** — large photos/illustrations with text as secondary
5. **Minimal / whitespace-heavy** — sparse elements, lots of breathing room
6. **Data-visual** — stats, metrics, ratings as visual elements alongside content
7. **Interactive / progressive disclosure** — expandable, hover-reveal, tabbed content

## Typography Variation

- Large display type as hero element vs. compact information-dense
- Dramatic size contrast (72px heading + 14px body) vs. even scale
- Serif headings + sans body vs. all sans-serif vs. monospace accents
- Tight letter-spacing vs. relaxed tracking

## Visual Weight Distribution

- **Top-heavy**: big visual element first, supporting text below
- **Bottom-heavy**: text intro, rich visual below
- **Centered focal point**: content radiates from center
- **Distributed**: even weight across the section

## How to Apply

For each generation session:

1. Pick 4 layout strategies that are maximally different
2. Assign a different compositional approach to each
3. Vary typography and visual weight across the 4
4. Write a brief plan noting what makes each variation distinct before generating code
5. Self-check: if you squint, could you tell the 4 apart? If not, increase contrast.
